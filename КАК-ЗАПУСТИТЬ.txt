SWIPD — маркетплейс (MVP)
============================

КАК ОТКРЫТЬ САЙТ:

1. Распакуй этот архив в любую папку.
2. Дважды кликни файл  start.bat
   - при первом запуске он сам установит зависимости (1–2 минуты);
   - затем поднимет локальный сервер.
3. Открой в браузере:  http://localhost:5173

Чтобы остановить — закрой чёрное окно (терминал).

ТРЕБОВАНИЯ:
- Установленный Node.js (LTS) — https://nodejs.org
  (на этом компьютере он уже стоит).

ЕСЛИ start.bat пишет, что Node не найден:
- открой новый терминал и выполни:
    cd путь\к\папке\nextgen-marketplace
    npm install
    npm run dev

ВНУТРИ ПРОЕКТА:
- src/        — весь код (экраны, компоненты, данные, логика)
- README.md   — описание и структура
- package.json — зависимости и команды

Telegram-уведомления о заказах настраиваются внутри приложения:
Профиль → Настройки → Кабинет продавца → «Заказы в Telegram».
